Definition 2.3.1.

The steady states of an autonomous DE are classified by their stability A steady state \(P\) is stable if \(p(t) \rightarrow P\) when the initial value is close to \(P\text{.}\) We can also call these steady states attractors. In the phase line diagram, the trajectories points toward such states. Similar, we can call a steady state stable or attractive from above or below if only one of the trajectories points towards the steady state. If both trajectories point away, the steady state is unstable.
