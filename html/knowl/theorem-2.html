Theorem 2.5.1.

In a equation of this form above, if \(F\) is continuous in both variables in an open set \(U \subset \RR^2\) and if \((x_0,y_0) \in U\text{,}\) then there exists \(\epsilon > 0\) such that the Initial Value Problem associated to the DE and the initial condition \(y(x_0) = y_0\) has a solution with domain at least \([x_0-\epsilon,x_0+\epsilon]\text{.}\)
