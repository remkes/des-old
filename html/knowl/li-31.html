Item.

We are informal with the constant. When we take exponents of each side of the equation, we should have multiplication by \(e^c\text{.}\) However, since this is still an undetermined constant, we simply write \(c\) instead of \(e^c\text{.}\) Also, when we drop the absolute value bars from \(y\text{,}\) we should have a \(\pm\) term. Again, since \(c\) can be either positive or negative, we don't worry about that \(\pm\text{.}\)
