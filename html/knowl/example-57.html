Example 5.6.1.
\begin{equation*} y^{\prime \prime} + y = 0 \hspace{2cm} y(0) = 1 \hspace{1cm} y^\prime(0) = 0 \end{equation*}

We apply the Laplace transform, making use of the initial values when we transform the derivative.

\begin{align*} \calL \{ y^{\prime \prime} + y \} \amp = 0\\ (s^2Y - sy(0) - y^\prime(0)) + Y \amp = 0\\ s^2Y - s + Y \amp = 0\\ (s^2+1)Y \amp = s\\ Y \amp = \frac{s}{s^2 + 1}\\ y \amp = \calL^{-1} \left\{ \frac{s}{s^2+1} \right\} = \cos t \end{align*}

We recover the expected \(y(t) = \cos t\text{,}\) but without any calculation of the characteristic equation or interpretation of complex roots. We also didn't have to get the complete solution and solve for unknown constants, since we made use of the initial values in the process. If initial values were not given, we would have to use unknown constants in their place in the calculation.
