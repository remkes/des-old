Example 5.1.4.

Lapalce transforms exist for piecewise-continuous functions with a certain exponential order. Here is a piecewise continuous function and its transform.

\begin{align*} f(t) \amp = \left\{ \begin{matrix} 0 \amp t \in [0,1]\\ t \amp t \in [1, \infty) \end{matrix} \right.\\ \calL \{ f(t) \} \amp = \int_0^\infty f(t) e^{-st} dt\\ \amp = \int_1^\infty te{-st} dt = \left. \frac{-te^{-st}}{s} \right|_1^\infty + \int_1^\infty \frac{e^{-st}}{s} dt\\ \amp = \frac{-e^{-s}}{s} + 0 + \left. \frac{-e^{-st}}{s^2} \right|_1^\infty = \frac{-e^{-s}}{s} - \frac{e^{-s}}{s^2} = \frac{-(s+1)e^{-s}}{s^2} \end{align*}

The Laplace transform is not a piecewise function, even thought the original was.
