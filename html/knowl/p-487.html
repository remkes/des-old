The unit step function \(u_a(t) f(t)\) cancels any part of \(f\) for \(t\lt a\text{,}\) but preserves \(f\) for \(t \geq a\text{.}\) To define a shift of \(f\text{,}\) we can write \(u_a(t) f(t-a)\) to remove anything before \(t=0\) from \(f\text{.}\) (Note that we're allowed piecewise continuous function for Laplace transforms.) We calculate to find the result of this shift in the \(t\)-domain.

\begin{align*} \calL \{ u_a(t) \} (s) \amp = \int_0^\infty u_a(t) e^{-st} dt = \int_a^\infty e^{-st} dt\\ \amp = \left. \frac{-e^{-st}}{s} \right|_a^\infty = \frac{e^{-as}}{s}\\ \calL \{ u_a (t) f(t-a)\} (s) \amp = \int_a^\infty e^{-st} f(t-a) dt\\ v \amp = t-a\\ \amp = \int_0^\infty e^{-s(v+a)} f(v) dv = e^{-sa} \int_0^\infty e^{-sv} f(v) dv\\ \amp = e^{-sa} \calL \{ f(t) \} (s) = e^{-sa} F(s) \end{align*}
