Then consider both the homoegenous equation \(Ly = 0\) and a non-homogenous equation \(Ly = Q(t)\text{.}\) If \(f\) is a solution to the non-homogenous equation, then \(Lf = Q(t)\text{,}\) and if \(g\) is a solution to the homogenous equation, then \(Lg = 0\text{.}\) By linearity, we can conclude that

\begin{equation*} L(f+\alpha g) = Lf + \alpha Lg = Q(t) + \alpha \cdot 0 = Q(t)\text{.} \end{equation*}
