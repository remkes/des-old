The loci we get are the two axes \(p=0\) and \(q=0\text{,}\) the horizontal line \(q = \frac{a}{b}\) and the vertical line \(p = \frac{d}{c}\text{.}\) There are four intersections points, \((0,0)\text{,}\) \(\left(\frac{d}{c},0\right)\text{,}\) \(\left(0,\frac{a}{b} \right)\) and \(\left( \frac{d}{c}, \frac{a}{b} \right)\text{.}\) The later is the only steady state with non-zero values, so the only possibility for a steady state which actually involves both species.
