This is an extension of the idea of analytic solutions. This solution is analytic if \(r \in \NN\text{.}\) Otherwise, it is very close, only differing by this multiple \((t-t_0)^r\text{.}\) If \(r\) is any negative real number, we have an asymptote at \(t=t_0\text{,}\) so we can't evaluate there. However, we expect convergence on \(0 \lt |t-t_0| \lt R\text{.}\) If \(r\) is a negative integer, we have a new name for these series.
