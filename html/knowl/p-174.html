Let \(L = \frac{d}{dt} + Q(t)\) be as before, and let \(y_h\) be a homogeneous solutions (a solution to \(Ly = 0\)). The technique we will use is called variation of parameters. This technique says we should look for a particular solution of the type \(y_p = g(t) y_h\text{.}\) Assuming that our solution has this special form, we have to try to find this \(g(t)\text{.}\) In order to do that, we put \(g(t) y_h\) into the equation.

\begin{align*} Ly_p(t) = L g(t) y(t) \amp = Q(t)\\ g^\prime(t) y(t) + g(t) y^\prime(t) + g(t) P(t) y(t) \amp = Q(t)\\ g^\prime(t) y(t) + g(t) \Big( y^\prime(t) + P(t) y(t) \Big) = g^\prime(t) + g(t) L y \amp = Q(t)\\ g^\prime(t) y(t) + g(t) 0 \amp = Q(t)\\ g^\prime(t) \amp = \frac{Q(t)}{y(t)}\\ g(t) \amp = \int \frac{Q(t)}{y(t)} \end{align*}
