This definition is a linear combination of Bessel functions of the first kind. For \(\nu \in \ZZ\text{,}\) this linear combination have been just a multiple of the one solution we know. To get around this, for \(\nu \in \ZZ\text{,}\) we define the Bessel function of the second kind as a limit.

\begin{equation*} Y_{\nu}(t) = \lim_{\alpha \rightarrow \nu} Y_{\alpha}(t) \end{equation*}
