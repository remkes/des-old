We will call these Laurent forms for \(P\) and \(Q\text{.}\) The coefficients \(p_1\) and \(q_2\) will be useful a bit later, so here are two useful identities to calculate \(p_1\) and \(q_2\text{.}\)

\begin{equation*} p_{-1} = \lim_{t \rightarrow 0 } t P(t) \hspace{2cm} q_{-2} = \lim_{t \rightarrow 0 } t^2 Q(t) \hspace{2cm} \end{equation*}
