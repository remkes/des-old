Solution 2.9.26.1.

This is a homogeneous equation, but it is a bit difficult to immediately see it. However, if I divide numerator and denominator by \(t\) on the right, the form will appear.

\begin{equation*} \frac{dy}{dt} = \frac{1}{1- \frac{y}{t}} \end{equation*}

I use the appropriate substitution: \(v = \frac{y}{t}\text{.}\) The expression on the left becomes \(f(v) = \frac{1}{1-v}\text{.}\) Then I can write the new differential equation in \(v\text{.}\)

\begin{equation*} \frac{dv}{dt} = \frac{f(v) - v}{t} = \frac{\frac{1}{1-v} - v}{t} = \frac{v - 1}{t(1-v)} = - t \end{equation*}

As expected, this is a separable equation. However, this is even better, since the dependent variable has entirely disappeared. I can solve this just by integrating both sides in \(t\)

\begin{equation*} v = \frac{-t^2}{2} + c \end{equation*}

Then I can reverse the substitution.

\begin{equation*} \frac{y}{t} = \frac{-t^2}{2} + c \end{equation*}

Finally, I can isolate \(y\) to produce the solution.

\begin{equation*} y = \frac{-t^3}{2} + ct \end{equation*}
