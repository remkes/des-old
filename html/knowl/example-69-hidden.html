This is an example with an integral.

\begin{align*} \frac{dy}{dt} + 6y(t) + 9 \int_0^t y(u) du \amp = -1 \hspace{1cm} y(0) = 0\\ sY+ 6Y + 9 \frac{Y}{s} \amp = \frac{-1}{s}\\ s^2 + 6sY + 9Y \amp = -1\\ Y \amp = \frac{-1}{(s+3)^2} = \frac{d}{ds} \frac{1}{s+3}\\ y \amp = te^{-3t} \end{align*}

We could try the previous example another way, by first differentiating both sides.

\begin{align*} \frac{dy}{dt} + 6y(t) + 9 \int_0^t y(u) du \amp = -1 \hspace{1cm} y(0) = 0\\ \frac{d^2y}{dt^2} + 6 \frac{dy}{dt} + 9y \amp = 0\\ r^2 + 6r + 9 \amp = 0 \implies r = 3\\ y \amp = ae^{-3t} + bte^{-3t}\\ y(0) \amp = 0 \implies a=0\\ y \amp = bte^{-3t} \end{align*}
