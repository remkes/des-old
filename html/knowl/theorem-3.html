Theorem 2.5.8.

If \(F\) is Lipschitz continuous in y and continuous (in the ordinary sense) in x on an open set \(U\) in \(\RR^2\) and if \((x_0,y_0) \in U\) then the initial value problem

\begin{align*} \amp \frac{dy}{dx} = F(x,y) \amp \amp y(x_0) = y_0 \end{align*}

has a unique solution \(y=f(x)\) defined on the domain \([x_0-\epsilon,x_0+\epsilon]\) for some \(\epsilon > 0\text{.}\)
