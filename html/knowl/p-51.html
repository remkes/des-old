A differentiable operator is an operator which acts on functions either by multplication by fixed functions or by differentiation. The operator is linear if the various pieces of the operation are added together. For example, a second order linear equation has this form

\begin{equation*} L = a(t) \frac{d^2}{dt^2} + b(t) \frac{d}{dt} + c \end{equation*}

(it is second order since the highest derivative is a second derivative).
