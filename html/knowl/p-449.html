We could redo the same work for negative \(\nu\text{,}\) but it is very similar (for \(n = \frac{-1}{2}\text{,}\) we would get a term \((2\nu - 1) c_1\text{,}\) so we would still conclude that \(c_1 = 0\text{,}\) hence all odd coefficients are \(0\)). In this way, we find the rest of the Bessel functions of the first kind.

\begin{equation*} J_{-\nu} = \sum_{n=0}^\infty \frac{(-1)^n}{n! \Gamma(1+n-\nu)} \left( \frac{t}{2} \right)^{2n-\nu} \end{equation*}
