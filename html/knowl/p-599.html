Recall that \(\imath\) is the complex number with the property \(\imath^2 = -1\text{.}\) Treating \(\imath\) as just an ordinary constant, prove that both \(e^{\imath t}\) and \(\cos t + \imath \sin t\) solve the differential equation \(y^{\prime \prime} + y = 0\) with initial conditions \(y(0) = 1\) and \(y^\prime(0) = \imath\text{.}\) Second order linear constant coefficient equations have unique solutions (which you can assume here, even though I haven't proved it in the course). Use this fact about unique solutions to conclude an alternate proof for Euler's formula. (8)

  2. Let \(a\text{,}\) \(b\text{,}\) and \(c\) be real constants and let

    \begin{equation*} L = a \frac{d^2}{dt^2} + b \frac{d}{dt} + c \end{equation*}

    be a second order linear differential operator in the variable \(t\text{.}\) (8)

    1. If a function \(f\) satisfies \(L f = 0\) and \(L (tf) = 0\text{,}\) prove that the function must satisfy the following first order equation.

      \begin{equation*} \frac{df}{dt} = \frac{-b}{2a} f(t) \end{equation*}

    2. Consider any second order linear constant coefficient homogeneous differential equation where the characteristic equation has a repeated root \(r\text{.}\) Verify that the initial solution \(f = e^{rt}\) satisfies the equation in the previous part of this question.
