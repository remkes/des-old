Before moving on to actual solution techinques, I want to talk about the second major interpretations of SOCCLDEs: alternating current circuits. The DE that will result is exactly the same as that for harmonic motion, but the interpretation of each of the coefficients is quite different. Instead of a position function \(x(t)\text{,}\) the system describes a charge function \(q(t)\text{;}\) the movement of that charge will be the alternating current. In this system, there are four components to a circuit: resistors, capacitors, inductors and an external electro-motive force. Let me describe these each in turn.
