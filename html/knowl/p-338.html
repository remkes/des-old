Let's set \(c_0 = 1\) and \(c_1 = 0\text{.}\) Then we use the recurrence relation to calculate some terms.

\begin{align*} c_0 \amp = 1\\ c_1 \amp = 0\\ c_2 \amp = \frac{-c_0}{(1)(2)} = \frac{-1}{2}\\ c_3 \amp = \frac{-c_1}{(2)(3)} = 0\\ c_4 \amp = \frac{-c_2}{(3)(4)} = \frac{1}{2\cdot 3 \cdot 4}\\ c_5 \amp = \frac{-c_3}{(4)(5)} = 0\\ c_6 \amp = \frac{-c_4}{(5)(6)} = \frac{-1}{2\cdot 3 \cdot 4 \cdot 5 \cdot 6 }\\ c_7 \amp = \frac{-c_5}{(6)(7)} = 0\\ c_8 \amp = \frac{-c_6}{(7)(8)} = \frac{1}{8!}\\ c_9 \amp = \frac{-c_7}{(8)(9)} = 0\\ c_{2n} \amp = \frac{(-1)^n}{(2n)!}\\ c_{2n+1} \amp = 0 \end{align*}
