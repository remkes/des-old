Example 5.9.4.

This is a ridiculous example with infinitely many \(\delta\)-functions, representing a new impact at every unit of time.

\begin{align*} f(t) \amp = \sum_{n=0}^\infty \delta_n (t)\\ y^{\prime \prime} + 3 y^\prime + 2y \amp = f(t) \hspace{1cm} y(0) = y^\prime(0) = 0\\ s^2 Y + 3sY + 2Y \amp = \frac{1}{1-e^{-s}} \int_0^1 e{-st} \delta_0 (t) dt = \frac{1}{1-e^{-s}}\\ Y \amp = \frac{1}{1-e^{-s}} \frac{1}{(s+2)(s+1)}\\ Y \amp = \left( \frac{1}{s+1} - \frac{1}{s+2} \right) \left( 1 _ e^{-s} + e^{-2s} + e^{-3s} + e^{-4s} + \ldots \right)\\ y \amp = e^{-t} - e^{-2t} + u_1(t) \left( e^{-(t-1)} - e^{-(2t-2)} \right) + u_2(t) \left( e^{-(t-2)} - e^{-(2t-4)} \right)\\ \amp \hspace{1cm} + u_3(t) \left( e^{-(t-3)} - e^{-(2t-6)} \right) + u_4(t) \left( e^{-(t-4)} - e^{-(2t-8)} \right)\\ y \amp = e^{-t} - e^{-2t} + u_1(t) \left( e^{1-t} - e^{2-2t} \right) + u_2(t) \left( e^{2-t} - e^{4-2t} \right)\\ \amp \hspace{1cm} + u_3(t) \left( e^{3-t} - e^{6-2t} \right) + u_4(t) \left( e^{4-t} - e^{8-2t} \right)\\ \amp = \sum_{k=0}^\infty u_k(t) \left( e^{k-t} - e^{2(k-t)} \right) \end{align*}

This is an interesting superposition of decay functions. They all have the same shape with a slightly higher peak. Adding them all up gives something which slowly rises higher and higher, while still trying to decay. Eventually the system does grow beyond any bounds, even with all the decay functions.
