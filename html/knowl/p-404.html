This is the solution for \(r=0\text{.}\) Now we proceed to the \(r=\frac{2}{3}\) case. (This is a non-integer exponent, so the solution is not a conventional Taylor series.)

\begin{align*} 3t \sum_{n=0}^\infty c_n \left(n + \frac{2}{3} \right) \left(n + \frac{2}{3}-1 \right) t^{n + \frac{2}{3}-2} + \sum_{n=0}^\infty c_n \left(n + \frac{2}{3} \right) t^{n + \frac{2}{3}-1} - \sum_{n=0}^\infty c_n t^{n + \frac{2}{3}} \amp = 0\\ t^{\frac{2}{3}} \left[ \sum_{n=0}^\infty 3c_n \left(n+ \frac{2}{3} \right) \left(n - \frac{1}{3} \right) t^{n-1} + \sum_{n=0}^\infty c_n \left(n+ \frac{2}{3} \right) t^{n-1} - \sum_{n=0}^\infty c_n t^{n} \right] \amp = 0\\ t^{\frac{2}{3}} \left[ \sum_{n=0}^\infty 3c_n \left(n + \frac{2}{3} \right) \left( n - \frac{1}{3} \right) t^{n -1} + \sum_{n=0}^\infty c_n \left(n + \frac{2}{3} \right) t^{n-1} - \sum_{n=1}^\infty c_{n-1} t^{n-1} \right] \amp = 0\\ t^{\frac{2}{3}} \left[ \left(3\frac{2}{3} \frac{-1}{3} + \frac{2}{3} \right) c_0 t^{-1} + \sum_{n=1}^\infty \left[ 3 c_n \left( n+ \frac{2}{3} \right) \left( n - \frac{1}{3} \right) + c_n \left( n + \frac{2}{3} \right) - c_{n-1} \right] t^{n-1} \right] \amp = 0\\ \left[ \left( n + \frac{2}{3} \right) \left( 3n - 1 + 1 \right) \right] c_n - c_{n-1} \amp = 0\\ c_n = \frac{c_{n-1}}{ \left( n + \frac{2}{3} \right) (3n)} = \frac{c_{n-1}}{(3n+2)n} \amp\\ c_{n+1} = \frac{c_n}{(3n+5)(n+1)} \amp \end{align*}
