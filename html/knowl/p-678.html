When we talked about the Lagrange polynomials, we derived a very nice orthogonality property for them. As opposed to the ordinary monomials \(x^n\text{,}\) the Lagrange polynomials were pair-wise orthogonal, using integration on \([-1,1]\) as the inner product. When we find a basis for a type ofseries, it is very convenient for integration if the basis functions are orthogonal. (Like in linear algebra, an orthogonal basis is always easy to use).
