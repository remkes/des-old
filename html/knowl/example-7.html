Example 2.1.11.
\begin{equation*} t \frac{dy}{dt} = 4y \end{equation*}

This is solved by \(y = ct^4\text{,}\) which is another reasonable family. However, there is a strange, singular solution.

\begin{equation*} y(t) = \left\{ \begin{matrix} -t^4 \amp t \leq 0 \\ t^4 \amp t>0 \end{matrix} \right. \end{equation*}

The derivatives of this function line up at the origin, so the function is of class \(C^1\) (actually \(C^3\text{,}\) if you work it out).
