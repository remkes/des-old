The \(X\) equation is a second order linear equation: \(X^{\prime \prime}(x) - \alpha X(x) = 0\text{.}\) If \(\alpha > 0\) then we expect exponential solutions \(X(x) = c_1 e^{\alpha x} + c_2e^{\alpha x}\text{.}\) However, the boundary conditions apply. \(X(0) = X(\pi) = 0\) implies that \(c_1 = c_2 = 0\text{;}\) the only solution here is the trivial solution \(X(x) = 0\text{.}\) Therefore, to get solutions that actually mean somthing, \(\alpha\) cannot be positive. If \(\alpha = 0\) then we get \(X = Ax + B\text{.}\) Again, the boundary conditions give \(A=B=0\text{,}\) so we only get the zero solution. Therefore, only \(\alpha \lt 0\) gives interesting solutions.
