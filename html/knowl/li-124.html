Item 2.

Let \(a\text{,}\) \(b\text{,}\) and \(c\) be real constants and let

\begin{equation*} L = a \frac{d^2}{dt^2} + b \frac{d}{dt} + c \end{equation*}

be a second order linear differential operator in the variable \(t\text{.}\) (8)

  1. If a function \(f\) satisfies \(L f = 0\) and \(L (tf) = 0\text{,}\) prove that the function must satisfy the following first order equation.

    \begin{equation*} \frac{df}{dt} = \frac{-b}{2a} f(t) \end{equation*}

  2. Consider any second order linear constant coefficient homogeneous differential equation where the characteristic equation has a repeated root \(r\text{.}\) Verify that the initial solution \(f = e^{rt}\) satisfies the equation in the previous part of this question.
