We have two cases. The odd terms all vanish, so the only place where we see \(c_1\) is the isolated term \(c_1 t\text{.}\) That implies that the linear polynomial \(c_1 t\) is a solution for any \(c_1\text{.}\) For the \(c_0\) terms, we get a series. Putting this together gives the general solution. Note that we have to start the series at \(n=2\text{,}\) since the pattern doesn't hold before that point.

\begin{equation*} y = c_0 \left[1 + \frac{t^2}{2} + \sum_{n=2}^\infty \frac{((-1)^n (2n-3)!}{2^{2n-2} n! (n-2)!} \right] + c_1 t \end{equation*}
