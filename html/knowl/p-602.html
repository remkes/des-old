  1. Consider this standard second order linear differential operator.

    \begin{equation*} L = \frac{d^2}{dt^2} + P(t) \frac{d}{dt} + Q(t) \end{equation*}

    Assume that \(t=0\) is an ordinary point for this differential equation. Write \(P(t)\) and \(Q(t)\) as series with unknown coefficients.

    \begin{align*} \amp P(t) = \sum_{n=0}^\infty p_n t^n \amp \amp Q(t) = \sum_{n=0}^\infty q_n t^n \end{align*}

    Using the process of solving by Taylor series at an ordinary point, calculate the coefficients of the solution up to \(c_4\text{.}\) (These coefficients will, of course, depend on the coefficients \(p_i\) and \(q_k\text{.}\) The expressions will get complicated quite quickly. The point of this exercise is to show how quickly the complexity grows then the coefficient functions need to be treated as series themselves.) (10)

  2. Use Taylor series centred at \(t=0\) to solved the following differential equation. (Make sure you verify that \(t=0\) is an ordinary point). (6)

    \begin{equation*} y^{\prime \prime} + 4t^2y = 0 \end{equation*}

  3. In the course notes, I solve the equation

    \begin{equation*} y^{\prime \prime} + ty = 0 \text{.} \end{equation*}

    In the previous question, you solved the equation

    \begin{equation*} y^{\prime \prime} + 4t^2y = 0 \text{.} \end{equation*}

    Even though this is only two examples, try to argue for a pattern for the solutions to the general equation (for \(\alpha \in \RR\text{,}\) \(\alpha \neq 0\) and \(n \in \NN\)).

    \begin{equation*} y^{\prime \prime} + \alpha t^ny = 0 \text{.} \end{equation*}

    What kind of behaviour do all these solutions display? (4)

  4. Use Taylor series centred at \(t=0\) to solve the following differential equation with initial values \(y(0) = 1\) and \(y^\prime(0) = 0\text{.}\) (Make sure you verify that \(t=0\) is an ordinary point). What radius do you expect for the solution to this differential equation? (8)

    \begin{equation*} (t^2 - 5t + 6) y^{\prime \prime} + y^\prime + ty = 0 \end{equation*}

    (You only need to calculate the solution up to the coefficient of \(x^7\text{,}\) since finding a pattern here is pretty difficult.)

  5. Determine which real numbers are ordinary points, singular points, and regular singular points for this differential equation. (4)

    \begin{equation*} y^{\prime \prime} + \frac{y^\prime}{(t-3)^2(t-4)(t-5)^3} + \frac{y}{(t-3)(t-4)^2(t-5)} = 0 \end{equation*}

  6. Determine which real numbers are ordinary points, singular points, and regular singular points for this differential equation. (4)

    \begin{equation*} y^{\prime \prime} + \frac{y^\prime}{\cos t - 1} + \frac{y}{\sin t - 1} = 0 \end{equation*}

  7. Determine which real numbers are ordinary points, singular points, and regular singular points for this differential equation. (4)

    \begin{equation*} y^{\prime \prime} + \frac{y^\prime}{\sqrt{t}} + \frac{y}{\sqrt{t^3}} = 0 \end{equation*}

  8. Consider the following differential equation. (10)

    \begin{equation*} ty^{\prime \prime} + \sinh t y = 0 \text{.} \end{equation*}

    Use the existence theorem for series solutions to prove that this DE has a solution centred at \(t=0\) with infinite radius of convergence. (That is, prove that there are no singular points.) Then solve the equation with initial conditions \(y(0) = \alpha\) and \(y^\prime(0) = 0\) where \(a \in \RR\text{.}\) (You only need to calculate the coefficients up to \(t^8\text{,}\) since finding a general solution here is very challenging.)

  9. Solve this differential equation using the Method of Frobenius centred at \(t=0\text{.}\) Find two independent solutions if you can; if you can't find one solution and comment on the form of the second solution. (8)

    \begin{equation*} 3t y^{\prime \prime} + (2-t) y^{\prime} - y = 0 \end{equation*}

  10. Solve this differential equation using the Method of Frobenius centred at \(t=0\text{.}\) Find two independent solutions if you can; if you can't find one solution and comment on the form of the second solution. (8)
