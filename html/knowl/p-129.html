Figure 2.4.3 shows the direction field and solution for this example. Notice the strange domain issues with this implicit plot. When \(|c| \leq 2\text{,}\) we have restricted domain solutions, represented by the closed curves. There are no solutions at all when \(c \lt -2\text{.}\) We have solutions with domain \(\RR\) only for \(c \geq 2\text{.}\) When \(c = 2\text{,}\) we get the strange crossed graph, which is not always differentiable. When \(c=-2\text{.}\) the “solution” is only defined at discrete points.
