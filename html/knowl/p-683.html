Let's express \(f(x) = x\) on \((-\pi, \pi)\) as a Fourier series. Notice that \(f(x)\) is odd, so only sine terms are expected. We calculate the coefficients of the sine terms with integrals.

\begin{align*} b_n \amp = \frac{1}{\pi} \int_{-\pi}^\pi f(x) \sin nx dx\\ \amp = \frac{1}{\pi} \int_{-\pi}^\pi x \sin nx dx\\ \amp = \frac{1}{\pi} \left. \left( \frac{-x\cos nx}{n} \right) \right|_{-\pi}^\pi + \int_{-\pi}^\pi \frac{\cos nx}{n} dx\\ \amp = \frac{1}{\pi} \left( \frac{ -\pi \cos n \pi }{n} + \frac{\pi \cos (-n\pi}{n}) + 0 \right)\\ \amp = \frac{1}{\pi} \left( \frac{-2\pi}{n} \right) \cos n\pi = \frac{-2(-1)^n}{n} = (-1)^{n+1}\ \frac{2}{n}\\ x \amp = 2 \sum_{n=1}^\infty \frac{(-1)^{n+1}}{n} \sin nx \end{align*}
