Definition 2.1.4.

A differentiable operator is an operator which acts on functions either by multplication by fixed functions or by differentiation. The operator is linear if the various pieces of the operation are added together. Fore example, a second order linear equation has this form

\begin{equation*} L = a(t) \frac{d^2}{dt^2} + b(t) \frac{d}{dt} + c \end{equation*}

(it is second order since the highest derivative is a second derivative).

I can use differentiable operators to simplify notations for DE. Using the operator \(L\) I just defined, a second order linear DE become simply \(Ly = f(t)\text{.}\)
