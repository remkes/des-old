\begin{equation*} (t^2-9) \frac{dy}{dt} + ty = 0 \end{equation*}

This is just a homogeneous DE. We note that \(t \neq \pm 3\) in the domain. The solution to the homogeneous case is

\begin{equation*} y = e^{-\int P(t) dt} = e^{-\int \frac{t}{t^2-9} dt} = e^{ - \frac{1}{2} \ln |t^2-9| + c} = \frac{c}{\sqrt{t^2-9}}\text{.} \end{equation*}

We should be careful with the absolute value in \(\ln |f(t)|\text{.}\) The calculation can be done in two pieces, when \(f(t) > 0\) and when \(f(t) \lt 0\text{.}\) In the previous calculation, we were a little careless with this detail.
