This example is from Roberts and Marion. It is technically a seperable equation, but it is still useful to see how the substitution technique works.

\begin{align*} \frac{dy}{dt} \amp = \frac{t}{y}\\ v \amp =\frac{y}{t}\\ f(v) \amp = \frac{1}{v}\\ \frac{dv}{dt} \amp = \frac{\frac{1}{v} - v}{t}\\ \int \frac{dv}{\frac{1}{v} -v} \amp = \int \frac{dt}{t}\\ \int \frac{v}{1-v^2} dv \amp = \ln |t| + c\\ \frac{-1}{2} \ln |1-v^2| \amp = \ln |t| + C\\ \frac{1}{\sqrt{1-v^2}} \amp = ct\\ \sqrt{1-v^2} \amp = \frac{c}{t}\\ 1-v^2 \amp = \frac{c}{t^2}\\ v^2 \amp = 1 - \frac{c}{t^2}\\ v \amp = \pm \sqrt{ 1 - \frac{c}{t^2}}\\ \frac{y}{t} \amp = \pm \sqrt{ 1 - \frac{c}{t^2}}\\ y \amp = \pm t \sqrt{1 - \frac{c}{t^2}} = \pm \sqrt{t^2 - c} \end{align*}

Notice that we reverse the substitution at the end, so that we end with the same variables as we started with.
