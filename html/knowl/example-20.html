Example 2.5.3.
\begin{equation*} y^\prime = \sqrt{|y|} \hspace{2cm} y(0) = 0 \end{equation*}

This \(F\) is also continuous near \((0,0)\text{,}\) so a solution exists by Peano's theorm. This IVP is solved by \(y=0\) and \(y = x^2/4\text{.}\) It's obvious that we need something more to ensure uniqueness.
