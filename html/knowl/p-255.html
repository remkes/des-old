The situation of repeated roots happens when \(b^2 = 4ac\text{,}\) so that the \(\pm\) disappears from the solutions to the quadratic. For harmonic systems, this only happens if the friction is exactly as this special value \(b = \sqrt{4ac}\text{.}\) This is the exact friction that allows for this one oscillation before exponential decay. We call these systems critically damped. This is the tipping point for friction: there is exactly enough friction to result in exponential decay.
